# AeroFit Customer Profiling & Product Recommendation Analysis

## Project Overview
This project analyzes customer purchase data from AeroFit to identify demographic and fitness-related characteristics associated with each treadmill model (KP281, KP481, KP781).

The goal is to build customer profiles and provide data-driven recommendations to support targeted marketing, product positioning, and sales strategy.

## Business Objective
- Identify who buys which AeroFit treadmill.
- Understand key factors influencing product choice.
- Recommend ideal target audiences for each product.
- Support marketing decisions using data.

## Tools Used
- Python
- Pandas, NumPy
- Datetime functions
- Matplotlib, Seaborn
